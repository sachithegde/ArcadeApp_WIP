The Arcade App is a Project which i created while learning CPP and SDL
While not all Best Practices or Designs have been Followed it Showcases the Following :
  1. Custom Shapes
  2. Custom Vector For Position, Rotation and Reflection
  3. Utilisation of SDL2
  4. Custom Collision Detection (Not Optimised)

This Repository will be reworked on in the future with more Features being added. 
Some things to Look Forward to 
  1. Custom Collisions (Optimised)
  2. Data Driven Scene Creations
  3. Tilemaps
  4. A Small Engine Like Structure that will allow for creating more games using the Classes and the Utility Provided
